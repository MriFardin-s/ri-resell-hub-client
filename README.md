# ♻️ RI Resell Hub

A modern full-stack resale marketplace where users can buy and sell second-hand products securely. The platform provides role-based dashboards for Buyers, Sellers, and Admins with Stripe payment integration, product moderation, wishlist management, and order tracking.

---

## 🚀 Live URL

**Client:** [https://client-url.vercel.app](https://ri-resell-hub-client.vercel.app/)

**Server:** [https://server-url.onrender.com](https://ri-resell-hub-server.vercel.app/)

---

## 🎯 Project Purpose

RI Resell Hub aims to provide a secure and user-friendly marketplace for buying and selling pre-owned products. It encourages sustainable shopping by giving used items a second life while ensuring safe transactions through authentication, product approval, and online payments.

## 📂 Installation

### Clone the repository

```bash
git clone https://github.com/MriFardin-s/ri-resell-hub.git
```

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```
